Layout Component

Layout Example


Example HTML

<div class="layout layout-main-left layout-stack-sm">

  <div class="col-md-9 col-sm-8 layout-main">
    <!-- CONTENT GOES HERE -->
  </div>
  
  <div class="col-md-3 col-sm-4 layout-sidebar">
    <!-- CONTENT GOES HERE -->
  </div>

</div>

Creating Sections


Section Classes

Size Class Description
Main .layout-main Add the class .layout-main to your main content column. (This will usually be the larger of the two columns.)
Sidebar .layout-sidebar Add the class .layout-sidebar to your main content column. (This will usually be the smaller of the two columns.)

Example HTML

<div class="col-md-9 layout-main"></div>

<div class="col-md-3 layout-sidebar"></div>

Column Sizing


You can size the layout columns using the default Bootstrap grid columns. You also have access to the responsive stacking of Bootstrap columns .col-md-* .col-sm-*

Separating Sections



Border Classes

Direction Class Description
Main Left .layout-main-left Add the class .layout-main-left to your .layout container. This will add the separating border to the right side of your main content column.
Main Right .layout-main-right Add the class .layout-main-right to your .layout container. This will add the separating border to the left side of your main content column.

Example HTML

<div class="layout layout-main-right">	
	<div class="col-md-3 layout-sidebar"></div>
	<div class="col-md-9 layout-main"></div>
</div>

Responsive Stacking


Stacking Classes

Size Class Description
Large .layout-stack-lg Collapses the two column layout @ widths below the Bootstrap large breakpoint.
Medium .layout-stack-md Collapses the two column layout @ widths below the Bootstrap medium breakpoint.
Small .layout-stack-sm Collapses the two column layout @ widths below the Bootstrap small breakpoint.
X-Small .layout-stack-xs Collapses the two column layout @ widths below the Bootstrap x-small breakpoint.

Example HTML

<div class="layout layout-main-left layout-stack-sm">

  <div class="col-sm-8 layout-main"></div>  
  <div class="col-sm-4 layout-sidebar"></div>

</div>

Thumbnails

Bootstrap Docs

Thumbnails Documentation

Thumbnail Image Hover

Image Hover is an extension of Bootstrap's thumbnail element. It easily allows you to have an icon to denote a zoom option when hovering over an image.

<div class="thumbnail-view">
	<a href="IMG_FULL_PATH" class="thumbnail-view-hover ui-lightbox"></a>
    <img src="IMG_THUMBNAIL_PATH" style="width: 100%" alt="Gallery Image">
</div>

Thumbnail Footer

The thumbnail footer provides an easy way to display comments, likes, and dates using icons.

<div class="thumbnail-footer">
	<div class="pull-left">
    	<a href="javascript:;"><i class="icon-thumbs-up"></i> 123</a> 
    	<a href="javascript:;"><i class="icon-thumbs-down"></i> 29</a>
    </div>

    <div class="pull-right">
    	<a href="javascript:;"><i class="icon-time"></i> 2 days ago</a>
    </div>
</div>

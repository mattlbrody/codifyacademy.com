Main Navigation

The main navigation work based on anchor links. To link the menu item to the specific block on the page, you need to add "#" + "id attribute" of that block to the link "href" attribute.

<nav>
<ul>
<li class="current-menu-item"><a href="#top">Top</a></li>
<li><a href="#portfolio">Portfolio</a></li>
<li><a href="#services">Services</a></li>

Fonts Setting up custom font

This theme use 1 custom free font called "Novecento wide Bold". You can download it here (require registration): http://www.fontspring.com/fonts/synthview/novecento-wide.

After downloading the font file, you need to generate font face kit of this font. You can do it here: http://fontface.codeandmore.com/ or on other such sites. After generating font face kit, you need to get 4 files of the font, and put them in the "fonts" folder of the template. These files must be named like (rename files with respecive extension if required):

  • novecentowide-bold.eot
  • novecentowide-bold.svg
  • novecentowide-bold.ttf
  • novecentowide-bold.woff

After you put the files into fonts folder, they will start working.
